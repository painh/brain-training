# 🧠 두뇌 트레이닝

닌텐도 DS 스타일의 두뇌 트레이닝 웹 앱입니다.

## 기능

- **계산 트레이닝** - 덧셈, 뺄셈 등 연산 문제를 풀며 두뇌를 단련
- **스도쿠** - 클래식 스도쿠 퍼즐
- **손글씨 숫자 인식** - TensorFlow.js 기반 숫자 인식 지원
- **학습 기록** - 캘린더에서 학습 이력 확인
- **두뇌 나이 측정** - 훈련 결과에 따른 두뇌 나이 계산
- **다국어 지원** - 한국어, 영어, 일본어

## 실행 방법

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
```

## 기술 스택

- React 19
- TypeScript
- Vite
- Zustand (상태 관리)
- TensorFlow.js (숫자 인식)
- Howler (사운드)
